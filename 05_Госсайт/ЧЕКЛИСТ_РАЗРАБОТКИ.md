# Чек-лист разработки госсайта под требования ФСТЭК

**Контекст:** сайт для госоргана с персданными → ГИС → Приказ ФСТЭК №117 (действует с 01.03.2026).
**Целевой класс (типовой):** К3 (низкий) + УЗ-3 по ПДн. Точный класс — из **акта классификации заказчика**.
**Для К3:** СЗИ 6 класса защиты, базовый уровень доверия.

> ⚠️ №117 убрал старые группы ИАФ/УПД/РСБ/АВЗ и жёсткий «базовый набор» (это был №17, отменён). Теперь XI групп мер + метрики КЗИ/ПЗИ + подбор под модель угроз. Ниже — практическая проекция требований на код сайта. Финальный список мер закрепляет модель угроз + методички ФСТЭК.
