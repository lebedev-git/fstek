# Пакет документов соискателя — чек-лист

По ПП РФ №79. Подаётся в ФСТЭК России.

## 1. Заявление

- [ ] Заявление о предоставлении лицензии с **явным перечислением выбранных видов работ** (пункты п.4 ПП №79).

## 2. Учредительные / регистрационные

- [ ] Устав ООО.
- [ ] Свидетельство/лист записи ЕГРЮЛ (ИНН, ОГРН).
- [ ] Документ о назначении руководителя.

## 3. Помещение

- [ ] Правоустанавливающие документы (собственность/аренда).
- [ ] Технический паспорт объекта информатизации (утв. руководителем).
- [ ] Приказ об определении границ контролируемой зоны.
- [ ] Приказ о вводе в эксплуатацию + назначение ответственных.
- [ ] Аттестат соответствия защищаемого помещения.
- [ ] Аттестат соответствия АРМ.

## 4. Персонал

- [ ] Копии дипломов (ИБ-образование) / удостоверений о переподготовке (≥360 ч).
- [ ] Документы, подтверждающие стаж (трудовые книжки/договоры).
- [ ] Штатное расписание, трудовые договоры, должностные инструкции.

## 5. Оборудование и ПО

- [ ] Документы права собственности/пользования на оборудование.
- [ ] Свидетельства о метрологической поверке.
- [ ] Лицензии на ПО, сертификаты на СЗИ (реестр ФСТЭК).

## 6. Документация и процессы

- [ ] Перечень нац. стандартов и методических документов под виды работ.
- [ ] Документы ограниченного доступа (ФСТЭК/Стандартинформ).
- [ ] Описание процесса работы с защищаемой информацией по форме ФСТЭК.
- [ ] Внутренние регламенты, положение о контроле качества.

## 7. Оплата

- [ ] Госпошлина за предоставление лицензии (реквизиты и размер — сверить по НК РФ на дату подачи).

---

## Частые причины отказа

- Виртуальный/номинальный адрес помещения.
- Неаттестованное помещение или АРМ.
- Формальный штат без реального подтверждения квалификации/стажа.
- Оборудование без действующей поверки.
- Несоответствие набора оборудования выбранным видам работ.
- Иностранное участие в ЮЛ.

## Следующий шаг

Определить состав видов работ (ЭТАП 0). Скажи, какие услуги планируешь оказывать клиентам — уточню обязательный комплект оборудования и точные требования к штату под них.
