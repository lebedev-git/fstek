# ФСТЭК №117 — Трекер соответствия

Веб-платформа для визуальной оценки готовности ГИС к требованиям **Приказа ФСТЭК России №117** (действует с 01.03.2026).

## Что делает

- **Карточки продуктов** — реестр проектов, каждый со своей карточкой (класс, УЗ, статус, балл). Создание через форму.
- **Дашборд проекта** — процент соответствия, разбивка по группам мер (V–XI + организация/инфра/аттестация).
- **Матрица мер** — интерактивная таблица: статус каждой меры, зона ответственности (разработчик/заказчик/совместно), доказательства. Статусы сохраняются в JSON.
- **Отчёт** — сводка: критические пробелы, задачи в работе, ручные проверки.
- **MCP-сервер** (`mcp/`) — подключаешь Claude → он анализирует код проекта и заносит оценки прямо в трекер. См. `mcp/README.md`.

## Два способа работы

1. **Вручную** — заводишь карточку, сам проставляешь статусы в матрице.
2. **Через MCP + Claude** — даёшь путь к проекту, Claude анализирует код и пакетно заносит оценки, дашборд обновляется.

## Запуск

```bash
cd platform
npm install
npm run dev
```

Открыть http://localhost:3000

## Структура

```
src/
  app/
    page.js            — дашборд
    matrix/            — матрица мер
    report/            — отчёт
    api/               — projects, assessment (CRUD оценок)
  components/          — Sidebar, ScoreRing, StatusBadge, MatrixClient
  data/
    measures.json      — меры №117 (источник истины)
    assessments.json   — статусы по проектам (сохраняется)
  lib/store.js         — чтение/запись, подсчёт баллов
```

## Модель данных

- `measures.json` — справочник мер: id, группа, зона ответственности, `autoCheckable` (задел под авто-сканер).
- `assessments.json` — оценки по проектам: статус (pass/fail/na/manual/todo), доказательство, заметка.

## Границы (честно)

- Это **трекер** (Уровень 1): статусы ставятся вручную. Балл = трекинг-готовность, **не** гарантия аттестации.
- Авто-сканер кода (Уровень 2) — следующая фаза; поле `autoCheckable` уже размечено.
- Финальный набор мер под конкретную систему закрепляет модель угроз + методички ФСТЭК.
